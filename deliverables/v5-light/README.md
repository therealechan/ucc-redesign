# UCC Redesign - v5 Light Mode Deliverable

## Overview
This is the v5 artifact for the UCC redesign project, specifically focusing on a clean, modern **light mode** theme. It provides a static, responsive frontend implementation of the landing page.

## File Structure
- `index.html`: The main HTML structure using semantic tags.
- `styles.css`: The styling file utilizing CSS variables for consistent theming, layout via CSS Grid/Flexbox, and responsive design.
- `README.md`: This documentation file.

## Design Highlights (Light Theme)
- **Backgrounds**: Pure white (`#ffffff`) for main background, slightly off-white (`#f9fafb`) for surface areas (hero, footer).
- **Text Contrast**: Dark slate text (`#111827`) for headings, softer gray (`#4b5563`) for body copy to ensure readability and WCAG contrast compliance.
- **Accents**: High-contrast primary blue (`#2563eb`) for buttons and links.
- **Borders & Shadows**: Subtle borders (`#e5e7eb`) and light shadows (`rgba(0,0,0,0.05)`) to create depth without dark harshness.

## How to View
To view this static landing page locally, you do not need a build process or server. Simply:
1. Navigate to this folder.
2. Open `index.html` directly in any modern web browser (Chrome, Firefox, Safari, Edge).

*Note: The page imports the Inter font from Google Fonts, so an internet connection is required to render the exact typography.*
